# Log File Analyzer

## Objective
This script analyzes web server log files to count the number of 404 errors, identify the most requested pages, and find the IP addresses with the most requests.

## How to Run
1. Ensure you have Python installed.
2. Place the log file (`access.log`) in the same directory.
3. Run the script: `python log_file_analyzer.py`.

## Output
The script will print a summary report showing:
- Total number of 404 errors.
- The most requested page.
- The IP address with the highest number of requests.

## Example
After running the script, you will see the total 404 errors, the top requested page, and the top IP address.
